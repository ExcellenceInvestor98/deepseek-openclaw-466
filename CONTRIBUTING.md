# Contributing

Thank you for improving DeepSeek OpenClaw.

## Development

1. Fork the repository.
2. Create a feature branch.
3. Make focused changes.
4. Run the project locally and verify the generated commands/config snippets.
5. Open a pull request with a clear description.

## Style

- Keep UI copy concise and practical.
- Never hardcode real API keys or user-specific secrets.
- Prefer small, typed utility functions over large inline logic blocks.
- Treat this project as an unofficial helper, not as an authoritative OpenClaw or DeepSeek implementation.
